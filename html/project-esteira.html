Projeto de Separação de Garrafas PET com Arduino

Descrição do Projeto:

Este projeto foi desenvolvido durante o ensino médio com o objetivo de criar uma esteira automatizada que separava garrafas PET por cor, utilizando materiais recicláveis. O projeto envolvia o uso de um Arduino e um solenoide para controlar a direção das garrafas, separando-as em dois compartimentos: um para garrafas verdes e outro para garrafas brancas.

Tecnologias e Ferramentas Utilizadas:

  • Arduino (Microcontrolador principal)
  • Sensores de cor (Para identificar as garrafas PET)
  • Solenoide (Para mudar a direção das garrafas)
  • Componentes eletrônicos diversos (LEDs, resistores, fios)
  • Materiais recicláveis (Para construção da esteira)

Funcionamento e Metodologia:

O projeto foi baseado em uma esteira motorizada onde as garrafas eram movidas uma a uma. Um sensor de cor, posicionado acima da esteira, identificava a cor da garrafa (verde ou branca). Após a identificação, um sinal era enviado ao Arduino, que controlava um solenoide responsável por desviar a garrafa para o compartimento correto.

Etapas de Implementação:

  1. Construção da Esteira: Utilizamos materiais recicláveis para criar a estrutura da esteira, e um motor foi acoplado para garantir o movimento contínuo das garrafas ao longo do sistema.
  2. Detecção de Cor: Um sensor de cor foi instalado na parte superior da esteira para identificar a cor das garrafas. Este sensor lia a cor e enviava um sinal para o Arduino, que tomava a decisão sobre o direcionamento.
  3. Controle do Solenoide: O Arduino recebia o sinal do sensor de cor e, com base nisso, acionava um solenoide que mudava a direção da garrafa. O solenoide redirecionava a garrafa para o lado correto, seja o compartimento de garrafas verdes ou brancas.

Comandos e Código Utilizado:

  • Exemplo de Código Arduino para Controle da Esteira: 
    
int sensorCorPin = A0; // Entrada do sensor de cor
int solenoidePin = 9;  // Controle do solenoide
int motorEsteiraPin = 10; // Controle do motor da esteira

void setup() {
  pinMode(solenoidePin, OUTPUT);
  pinMode(motorEsteiraPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int cor = analogRead(sensorCorPin); // Lê o valor do sensor de cor

  // Lógica para determinar a cor da garrafa
  if (cor > 500) {
    // Garrafa branca
    Serial.println("Garrafa Branca Detectada");
    digitalWrite(solenoidePin, LOW);  // Solenoide não aciona, garrafa vai para o compartimento branco
  } else {
    // Garrafa verde
    Serial.println("Garrafa Verde Detectada");
    digitalWrite(solenoidePin, HIGH); // Solenoide aciona, garrafa vai para o compartimento verde
  }

  // Mantém a esteira ligada
  digitalWrite(motorEsteiraPin, HIGH);
  delay(1000);  // Aguarda 1 segundo para próxima garrafa
}

Desafios Enfrentados:

O maior desafio foi garantir a precisão na detecção de cor, especialmente em ambientes com iluminação variável. Ajustes na calibração do sensor foram necessários para garantir que ele pudesse diferenciar corretamente entre garrafas verdes e brancas. Além disso, a integração do solenoide com o sistema de controle do Arduino exigiu um refinamento nos tempos de resposta para garantir que o redirecionamento das garrafas fosse feito corretamente.

Resultados:

O projeto foi bem-sucedido em separar as garrafas por cor, utilizando um sistema totalmente automatizado e feito com materiais recicláveis. O uso do Arduino permitiu um controle eficiente do solenoide e da esteira, garantindo a separação precisa de garrafas PET verdes e brancas. Esse projeto foi uma excelente introdução ao uso de microcontroladores e sistemas de automação.
