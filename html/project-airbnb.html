Projeto Identificação de Melhores Preços do Airbnb

Descrição do Projeto:

Este projeto foi desenvolvido para identificar as melhores opções de hospedagem no Airbnb, levando em consideração tanto os preços quanto as avaliações qualitativas dos usuários. Utilizando tecnologias de web scraping, foi possível coletar e organizar dados de imóveis para auxiliar na tomada de decisão sobre o melhor lugar para se hospedar durante uma viagem.

Tecnologias e Ferramentas Utilizadas:

  • Python (Linguagem de programação principal)
  • BeautifulSoup (Web scraping)
  • Requests (Requisições HTTP)
  • Pandas (Organização e análise de dados)
  • Matplotlib e Seaborn (Visualização de dados)
  • VSCode (Ambiente de desenvolvimento)

Metodologia:

O projeto foi estruturado em três etapas principais:

  1. Coleta de dados: Utilizamos web scraping para extrair dados diretamente das páginas de anúncios do Airbnb, incluindo preços, localização e avaliações dos usuários.
  2. Análise dos dados: Após a coleta, os dados foram organizados e limpos usando a biblioteca Pandas, permitindo análises detalhadas.
  3. Visualização: Por fim, gráficos comparativos foram gerados com Matplotlib e Seaborn, facilitando a comparação entre os imóveis.

Desafios Enfrentados:

Contornar as limitações de requisições do Airbnb foi um desafio. Para resolver esse problema, implementamos intervalos de tempo entre as requisições, simulando a navegação humana.

Resultados:

Conseguimos construir um sistema de recomendação que ajuda a identificar os melhores imóveis para hospedagem, baseado tanto no preço quanto na avaliação dos hóspedes. Os resultados foram apresentados de forma visual e clara, ajudando o usuário a tomar decisões rápidas e informadas.
