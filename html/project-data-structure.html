Projeto de Estruturas de Dados em Java e C

Descrição do Projeto:

Ao longo da faculdade, desenvolvi diversos projetos relacionados a estruturas de dados utilizando as linguagens de programação Java e C. Esses projetos abrangeram desde conceitos simples, como pilhas e filas, até estruturas mais avançadas, como árvores binárias de busca e listas encadeadas. Abaixo, detalho os principais conceitos de cada estrutura e como foram implementados em ambas as linguagens.

Tecnologias e Ferramentas Utilizadas:

  • Java (Linguagem de Programação)
  • C (Linguagem de Programação)
  • NetBeans (Ambiente de desenvolvimento para Java)
  • GCC (Compilador para C)
  • VSCode (Ambiente de desenvolvimento para C)

Conceitos de Estruturas de Dados e Implementação:

  1. Pilhas (Stacks):

    2. Uma pilha é uma estrutura de dados que segue o princípio LIFO (Last In, First Out). O último elemento a ser inserido é o primeiro a ser removido. Implementei a estrutura de pilha tanto em Java quanto em C. No projeto, as operações básicas foram implementadas:

    • Push: Adiciona um elemento ao topo da pilha.
    • Pop: Remove o elemento do topo da pilha.
    • Peek: Exibe o elemento do topo sem removê-lo.
    • isEmpty: Verifica se a pilha está vazia.
    Em Java, utilizei classes e objetos para gerenciar a pilha, enquanto em C, a pilha foi gerenciada utilizando arrays e ponteiros.

  2. Filas (Queues):

    3. Uma fila segue o princípio FIFO (First In, First Out). O primeiro elemento a entrar é o primeiro a sair. Nas implementações, foram desenvolvidas as seguintes operações:

    • Enqueue: Insere um elemento no final da fila.
    • Dequeue: Remove o primeiro elemento da fila.
    • Front: Mostra o primeiro elemento sem removê-lo.
    Assim como nas pilhas, em Java utilizei classes e objetos, enquanto em C, as filas foram gerenciadas com o uso de arrays e ponteiros.

  3. Listas Encadeadas (Linked Lists):

    4. As listas encadeadas são estruturas de dados compostas de nós, onde cada nó contém um valor e uma referência para o próximo nó. Desenvolvi tanto listas simplesmente encadeadas quanto duplamente encadeadas, com as operações:

    • Inserção: Insere um novo nó no início, no final ou em uma posição específica da lista.
    • Remoção: Remove um nó específico da lista.
    • Busca: Busca por um valor específico dentro da lista.
    Em C, as listas foram implementadas com o uso de ponteiros para os nós, e em Java, as listas foram gerenciadas com a criação de classes para os nós e a lista.

  4. Árvores Binárias (Binary Trees):

    5. Árvores binárias são estruturas hierárquicas em que cada nó possui até dois filhos. Implementei árvores binárias de busca (BST), que têm a propriedade de que o valor à esquerda de um nó é sempre menor, e o valor à direita é sempre maior. As operações incluíram:

    • Inserção: Insere um novo nó na árvore.
    • Busca: Procura por um valor específico na árvore.
    • Remoção: Remove um nó da árvore.
    • Percursos: Realizei percursos em Pré-Ordem, Em-Ordem e Pós-Ordem.
    As árvores foram implementadas utilizando classes em Java e estruturas com ponteiros em C, com destaque para a eficiência de busca e inserção em árvores binárias balanceadas.

  5. Tabelas Hash (Hash Tables):

    6. As tabelas hash permitem armazenar e buscar dados de forma eficiente, utilizando uma função hash para mapear chaves a posições em um array. Implementei tanto em Java quanto em C, utilizando:

    • Função Hash: Função para transformar a chave em um índice do array.
    • Tratamento de Colisões: Utilizei o método de encadeamento para resolver colisões.
    A implementação foi focada em inserir, buscar e remover elementos com alta eficiência.

Desafios Enfrentados:

O maior desafio foi garantir a correta manipulação de ponteiros em C, especialmente nas operações de remoção em listas encadeadas e árvores binárias. Em Java, a complexidade estava na implementação das interfaces e na reutilização de código entre diferentes estruturas. Outro ponto crítico foi garantir que todas as operações fossem realizadas com eficiência de tempo e espaço.

Resultados:

Esses projetos resultaram em um profundo entendimento das diferentes estruturas de dados, seus usos e sua eficiência. Além disso, a experiência de implementar as mesmas estruturas tanto em Java quanto em C me proporcionou uma visão clara das vantagens e desafios de ambas as linguagens em termos de gerenciamento de memória e abstração.
