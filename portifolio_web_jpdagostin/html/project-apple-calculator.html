Projeto Calculadora em React Native

Descrição do Projeto:

Este projeto foi desenvolvido como parte de um trabalho de faculdade e tem como objetivo recriar uma calculadora simples em React Native, inspirada no design da calculadora da Apple. A interface gráfica segue o padrão visual icônico da Apple, utilizando os botões e layout semelhantes ao do iOS.

Tecnologias e Ferramentas Utilizadas:

  • React Native (Framework para desenvolvimento móvel)
  • JavaScript (Linguagem de programação principal)
  • Expo (Plataforma para execução e desenvolvimento de apps React Native)
  • VSCode (Ambiente de desenvolvimento)

Funcionalidades Implementadas:

A calculadora implementa as funcionalidades básicas de uma calculadora comum, permitindo:

  • Adição, subtração, multiplicação e divisão
  • Limpar o valor atual (C)
  • Uso do botão de porcentagem
  • Alternar entre valores positivos e negativos

Metodologia:

O projeto foi estruturado da seguinte forma:

  1. Desenvolvimento da Interface: Utilizando componentes do React Native, a interface foi construída com botões que replicam o visual da calculadora da Apple, com atenção ao espaçamento e funcionalidade de cada botão.
  2. Lógica de Cálculo: A lógica de operações matemáticas foi implementada utilizando JavaScript, de forma a garantir precisão nos cálculos e resposta imediata nas operações.
  3. Testes e Ajustes: Após o desenvolvimento inicial, foram realizados testes no Expo para garantir que a calculadora funcionasse corretamente em diferentes dispositivos móveis.

Desafios Enfrentados:

Um dos principais desafios foi adaptar o layout da calculadora para funcionar corretamente em diferentes tamanhos de tela, especialmente mantendo a estética fiel à calculadora da Apple em dispositivos de tamanhos variados.

Resultados:

O projeto final conseguiu replicar com sucesso uma calculadora funcional e estética, semelhante à calculadora do iOS. A usabilidade foi testada em diferentes dispositivos móveis, proporcionando uma experiência suave e intuitiva para os usuários.
